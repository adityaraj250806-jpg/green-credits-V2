## 🚀 The Vision
Waste management is a critical challenge. **GreenCredits** solves this by turning civic duty into a rewarding experience. Citizens report unmanaged waste, field workers resolve the issues, and users earn "Green Credits" to unlock badges and real-world recognition. It's a complete ecosystem for a cleaner India.

## ✨ Platform Highlights

### 🛡️ For Citizens (Gamification)
- **Report & Earn:** Upload photos of waste to earn Green Credits.
- **AI Verification:** Smart digital fingerprinting (MD5) prevents users from uploading duplicate or fake photos.
- **Rankings:** Unlock Bronze, Silver, Gold, and Platinum badges based on civic contribution.

### 🗺️ For the Public (Transparency)
- **Live Impact Heatmap:** Interactive map built with `Leaflet.js` showing real-time waste hotspots and cleaned areas across the city.

### ⚡ For Administrators (God Mode)
- **Real-Time Dashboard:** A WebSocket-powered (`Socket.io`) command center. As soon as a citizen reports waste, the admin dashboard updates instantly—no page refresh needed.
- **Worker Management:** Assign specific cleanup tasks to registered ground workers.

## 🛠️ Run it Locally
Want to test the code on your own machine? It's simple:
```bash
# 1. Clone the repository
git clone https://github.com/adityaraj250806-jpg/green-credits-V2.git

# 2. Install dependencies
cd green-credits-V2
npm install

# 3. Setup Environment Variables
# Create a .env file and add your MONGODB_URI and CLOUDINARY_URL

# 4. Start the server
npm run dev
```
